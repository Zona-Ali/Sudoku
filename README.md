# 🧩 Sudoku – Game & Solver

Sudoku is a full-stack web application that allows users to play Sudoku puzzles and automatically solve a given sudoku using a built-in solving algorithm.
The project features an interactive UI, difficulty levels, timer tracking, and an intelligent backtracking-based solver.

---

## ✨ Features

### 🎮 Sudoku Gameplay
- Interactive 9×9 Sudoku board
- Multiple difficulty levels
- Real-time input validation
- Reset and New Game functionality
- Timer tracking

### 🤖 Sudoku Solver
- Automatically solves any valid Sudoku puzzle
- Uses Backtracking Algorithm
- Validates board before solving
- Instantly displays completed solution

### 🧠 Game Logic
- Ensures valid moves (no duplicates in rows, columns, or 3×3 boxes)
- Dynamic puzzle generation
- Error handling for invalid inputs

### 🎨 UI & UX
- Responsive design
- Clean and minimal interface
- Smooth user interaction
- Mobile-friendly layout
